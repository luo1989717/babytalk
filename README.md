《看图说字》2岁宝宝语言启蒙练习手册需求文档

一、项目背景

面向2岁左右宝宝，设计一本“看图说字”的简单练习手册。宝宝处在口腔探索、语言模仿和词汇快速积累阶段，可能愿意说话，但发音不稳定、不清楚。手册目标不是纠正发音，而是通过亲子互动、图片识别、简单模仿，鼓励宝宝主动开口、表达需求和建立词物对应关系。

参考原则：CDC提到2岁儿童通常会开始组合简单词语，如“more milk”；ASHA也建议家长通过日常互动、重复、歌曲和简单语言支持2-3岁儿童沟通发展。
来源：CDC 2岁里程碑、ASHA 2-3岁沟通里程碑

二、目标用户

使用者：2岁左右男宝宝，适合18-30个月儿童延展使用。
陪伴者：父母、祖辈、早教老师、育儿陪伴者。
使用场景：睡前、饭后、亲子阅读、早教游戏、出门前后的短时间互动。

三、产品目标

帮助宝宝建立“图片-实物-声音-词语”的连接。
鼓励宝宝模仿简单单字、叠词和生活高频词。
增强宝宝主动表达需求的机会，例如“抱”“吃”“喝”“车”。
降低家长教学压力，提供简单、温和、可重复的引导话术。
让练习像游戏，不像考试。
四、非目标

不作为语言发育诊断工具。
不要求宝宝发音标准。
不进行复杂拼音、识字或笔画教学。
不设计长句阅读任务。
不用“对/错”强反馈打击宝宝表达意愿。
五、内容设计原则

每页只练一个核心字或一个核心词。
图片要真实、清晰、颜色明亮，避免复杂背景。
文字大、少、醒目，优先使用单字、叠词、生活词。
每个词配一个动作或互动任务。
家长话术短句化，比如“这是车。车车。宝宝说：车。”
接受近似发音，比如“车”说成“得”“呃”，也给予积极回应。
每次练习控制在5-10分钟，避免疲劳。
六、手册规格建议

名称：《宝宝看图说字》
尺寸：方形或横版，建议170mm x 170mm 或 A5横版。
页数：24-36页。
材质：圆角厚纸板或覆膜铜版纸，耐翻、耐擦。
安全：无小零件、无尖角、无易脱落贴片；考虑宝宝啃咬行为，油墨和覆膜需符合儿童用品安全要求。

七、页面结构

每个练习页包含：

大图：占页面60%-70%。
大字：1个核心字，字号大。
亲子提示：一句简短引导语。
动作互动：指一指、摸一摸、学一学、找一找。
可选扩展词：1个相关叠词或短语。
示例页面：

核心字：车
图片：红色小汽车
家长话术：这是车，车车开来了。
互动：宝宝指一指车轮。
扩展：车车、开车、滴滴。

八、词汇选择

第一阶段：最容易模仿的亲密称呼和需求词
爸、妈、抱、吃、喝、拿、要、走、睡、看

第二阶段：生活常见物品
球、车、灯、鞋、帽、杯、门、床、书、包

第三阶段：动物和声音模仿
猫、狗、鸭、牛、鱼、鸟、羊
可配音效：喵、汪、嘎、哞、咩

第四阶段：动作词
跑、跳、拍、吹、推、拉、坐、洗、穿、开

第五阶段：情绪和社交
笑、哭、好、怕、疼、拜、给、谢、亲

十、互动玩法

看一看：宝宝看图片，家长说名称。
指一指：家长问“车在哪里？”
学一学：家长说“车车，滴滴”，宝宝模仿任意声音。
做一做：看到“拍”，一起拍拍手。
找一找：在家里找同类物品，比如“杯杯在哪里？”
选一选：两张图中让宝宝选“球”。
说一说：宝宝愿意开口时，不打断、不急着纠音。
十一、视觉要求

图片风格：温暖、简洁、真实感强，可用手绘但必须易识别。
背景：浅色干净背景。
颜色：高对比但不刺眼。
人物：如果出现宝宝或家长形象，表情友好自然。
文字：黑色或深色大字，避免花体字。
版面：每页信息极少，留白充足。

十二、家长使用提示

宝宝说不准是正常现象，重点是“愿意说”。
不要频繁说“错了”，可以重复正确发音示范。
宝宝只指图、不说话时，也算参与。
每个词可以重复很多天，不需要追求进度。
可以把宝宝发出的近似音接住，比如宝宝说“汪”，家长回应“对，狗狗汪汪”。
十三、验收标准

2岁宝宝能独立翻看并对图片产生兴趣。
每页核心词清晰，不需要成人额外解释图片是什么。
家长能在30秒内理解每页怎么玩。
手册连续使用一周后，宝宝愿意指认、模仿声音或尝试说词。
整体无考试感、无压迫感，亲子互动自然轻松。
十四、备注

如果宝宝到了2岁后长期几乎没有有意义的词、很少回应名字、听不懂简单指令，或家长明显担心听力和沟通发展，建议咨询儿科医生、儿童保健医生或语言治疗师。手册用于日常启蒙和亲子陪伴，不替代专业评估。
